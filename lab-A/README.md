Лабораторная работа А: Связные списки

Вариант 11: Очередь (1)
Задание:
Реализуйте очередь с помощью связного списка
Реализуйте очередь с помощью массива
Сравните полученные реализации, в чем преимущества и недостатки каждой?

Реализация очереди с помощью связного списка
Преимущества:
Нет ограничения на размер очереди (кроме доступной памяти).
Операции добавления и удаления выполняются за O(1).

Реализация очереди с помощью массива
Преимущества:
Простая структура данных, легко реализуется.
Высокая локальность данных — элементы расположены подряд в памяти, что ускоряет доступ.
Недостатки:
Необходимость заранее задавать размер массива.
При переполнении нужно перераспределять память или отказывать во вставке.
